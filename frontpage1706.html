Work Assist: Intelligent Enterprise Assistant

We implemented a robust and scalable solution using Microsoft Phi-3.5-mini-Instruct for chatbot and document processing. This model was chosen for its superior resource efficiency, faster latency, and excellent handling of large-scale organizational queries, particularly in multilingual contexts such as Hindi and Hinglish.

Our frontend leverages the power of Streamlit, offering a modern and scalable user interface, capable of supporting thousands of simultaneous connections, with real-time updates to enhance the user experience.

Fine-tuning was done using Hugging Face embeddings to ensure the chatbot provides accurate and contextually relevant responses. We've also optimized performance, achieving an average response time of 4.2 seconds after processing 50 queries.

For security, we implemented Two-Factor Authentication (2FA) using Yagmail and Redis, along with secure data storage via PostgreSQL. Additionally, a bad language filter ensures professional and respectful communication.
